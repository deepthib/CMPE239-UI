Freeway Traffic Prediction

Problem

As congestion is increasing on the freeways and hence the delays, there is an increasing need for a system which provides accurate and reliable travel time prediction. The focus in this paper is mainly on predicting the time to commute between any two points of a freeway for any future departure time. These daysâ€™ freeways are equipped with sensors and loop detectors to collect data in order to give current travel time to commuters. The data is collected and processed over the internet in order to give outputs like how much time it will take for the commuter to travel from one location to another at that very moment. Data is processed by using some decision making algorithm whose output is time to travel between two locations at that very moment and is displayed to commuters.

Technology Stack

1. Pandas ( Data cleaning ) : Pandas is a python library used for data manipulation. We used this library in our project to cleanse the data

2. NuPIC ( Data prediction ) : NuPIC is a set of tools which includes machine learning algorithms that enable self training and pattern recognition. We have used swarming in NuPIC to predict models.

3. MatPlot ( Plotting ) : MatPlot is a python library used for plotting.

Implementation

Data is collected from the sensors on the freeway. We have data in form of csv file. We have data for the year 2011 for the entire New York city.The data in csv file has the following columns : RCStation, Start_Time, Direction, Lane and Count. We only selected monitoring stations that has consecutive hourly count data for more than 60 days for further analysis. For those stations, we picked one direction and aggregate all traffic count from all lanes, and then select the longest continuous recording segment. We used Cortical Learning Algorithm (CLA) in the Numenta Platform for Intelligent Computing (NuPIC) to train the traffic data. Predictions are made based on this.
